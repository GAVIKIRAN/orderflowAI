How RFID scanning works in OrderFlow
RFID and barcode scanning is optional — you can place orders directly via chat. When you do scan a tag, it will:
  1. Look up the matching inventory item by RFID tag
  2. Reduce stock by 1 unit (consumption event)
  3. Check if stock has reached the reorder threshold
  4. If yes — auto-create a purchase order for that item
  5. Play an audio confirmation beep
